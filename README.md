# Bank-Management-System-C-OOP
This is a C++ Object-Oriented Programming based Bank Account Management System that simulates real banking operations. It allows users to create Savings and Current accounts, securely log in using PIN authentication, and perform actions like deposit, withdrawal, balance inquiry, and money transfer between accounts. The system also includes an admin panel for managing all accounts, including viewing, deleting, unlocking accounts, and running monthly processing. It uses file handling to save and load data, ensuring all account information is stored permanently. The project demonstrates key OOP concepts such as inheritance, polymorphism, and abstraction, along with real-world system design and problem-solving skills.
